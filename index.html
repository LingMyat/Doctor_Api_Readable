Get Days | GET http://127.0.0.1:8000/api/days
Description

Example
{
"id": 1 ,

}

Auth

Register | POST http://127.0.0.1:8000/api/v1/doctor/auth/register
Description
Doctor register.
Body
{
"name": "User" ,
"email_or_phone": "user@gmail.com" ,
"password": "password" ,
"confirm_password": "password"
}
Login | POST http://127.0.0.1:8000/api/v1/doctor/auth/login
Description
Doctor login.
Body
{
"email_or_phone": "user@gmail.com" ,
"password": "password"
}
Get Profile | GET http://127.0.0.1:8000/api/profile
Description
This API will return current Token owner. To perform this action, user must be authenticated.
Update | POST http://127.0.0.1:8000/api/v1/doctor/auth/update
Description
Doctor Profile Update. To perform this action, user must be authenticated.
Body
{
"name": "User Name" ,
"email": "user@gmail.com" ,
"phone": "09123456789" ,
"degree": "user degree" ,
"SAMA": "user SAMA" ,
"biography": "user biography" ,
"professions": [
{ sub_diagnosis_id } , ...
]
}
Logout | DELETE http://127.0.0.1:8000/api/v1/doctor/auth/logout
Description
Patient Logout.To perform this action, user must be authenticated.

Doctor Time

Store Doctor Time | POST http://127.0.0.1:8000/api/doctor-time
Description
Store doctor times.
body
{
"time_from": "08:30" ,
"time_to": "12:00" ,
"day_id": 4
}
Update Doctor Time | PATCH http://127.0.0.1:8000/api/doctor-time/{doctor_time_id}
Description
Update doctor times.
body
{
"time_from": "08:00" ,
"time_to": "11:30" ,
"day_id": 1
}
Delete Doctor Time | DELETE http://127.0.0.1:8000/api/doctor-time/{doctor_time_id}
Description
Update doctor times.
Example
http://127.0.0.1:8000/api/doctor-time/1

Diagnosis

Main Diagnosis
Get All Main Diagnosis | GET http://127.0.0.1:8000/api/v1/main-diagnosis
Description
Fetch all main diagnosis from the database. !! Doctor Specilist are only sub_diagnosis !!
Example
{
"id": 1 ,
"name": "ကင်ဆာရောဂါများ" ,
"sub_diagnosis": [
{
"id": 8 ,
"name": "Can Cer" ,
"image": "something.jpg / something.png"
}
]
}
Get Specific Main Diagnosis | GET http://127.0.0.1:8000/api/v1/main-diagnosis?id={id}
Description
Fetch main diagnosis by ID from the database.
Body
"http://127.0.0.1:8000/api/v1/main-diagnosis?id=6"

Appointments

Check All Appointments | GET http://127.0.0.1:8000/api/v1/doctor/appointments
Description
Fetch all own appointment from the database. To perform this action, user must be authenticated.
Example
{
"id": 8 ,
"status": "success" ,
"time": "09:40 AM" ,
"day": [
{
"name": "Monday" ,
"name_mm": "တနင်္လာနေ့"
}
] ,
"diagnosis": [
{
"id": 8 ,
"name": "Can Cer" ,
"image": "something.jpg / something.png"
}
] ,
"patient": [
{
"id": 1 ,
"image": "something.jpg / something.png" ,
"name": "Ling Myat Aung" ,
"Gender": "Male" ,
"phone": "09459192584"
"address": "Mandalay"
"age": 19
}
]
}
Update Appointment(success/canceled) | PATCH http://127.0.0.1:8000/api/v1/doctor/appointments/{appointment_id}
Description
Update an appointment. To perform this action, user must be authenticated.
Body

Success

{
"status": "success"
}

Canceled

{
"status": "canceled" ,
"cancel_remark": "cancel_remark(Text)"
}

Chat Room

Get All Chat Rooms | GET http://127.0.0.1:8000/api/v1/rooms
Description
Fetch all chat rooms from the database.
Example
{
"id": 1 ,
"name": "Traditional Medicine" ,
"room_img": "something.jpg / something.png"
}
Get All Chat Messages From Specific Rooms | Get http://127.0.0.1:8000/api/v1/rooms/{room_id}/room-messages
Description
Fetch all messages from specific_rooms. Room_id can get from "Get All Chat Rooms" . To perform this action, user must be authenticated.
Example
"http://127.0.0.1:8000/api/v1/patient/rooms/1/room-messages"
{
"room": {
"id": 1 ,
"name": "Traditional Medicine" ,
"room_img": "something.jpg / something.png"
} ,
"messages": [
{
"id": 1 ,
"message": "Message Text(can be long text)" ,
"sender": {
"id": 1 ,
"name": "Ling Myat Aung" ,
"image": "something.jpg / something.png" ,
"role": "can be (Patient or Doctor)"
} ,
"child_messages": [
{
"id": 2 ,
"message": "Message Text(can be long text)"
}
]
}
] ,
"last_sender": {
"id": 1 ,
"name": "Ling Myat Aung" ,
"image": "something.jpg / something.png" ,
"role": "can be (Patient or Doctor)"
}
}
Send Message | POST http://127.0.0.1:8000/api/v1/rooms/store-room-message
Description
Send message to room. Store messages. To perform this action, user must be authenticated.
Body
{
"room_id": 1 ,
"message": "Lorem Ipsum is simply dummy text of the printing and typesetting industry."
}
Send Image | POST http://127.0.0.1:8000/api/v1/rooms/store-chat-image
Description
Send image to room. Store chat image. To perform this action, user must be authenticated.
Body
{
"room_id": 1 ,
"image": "something.jpg / something.png"
}
